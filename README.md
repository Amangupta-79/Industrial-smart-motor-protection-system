# Industrial Smart Motor Protection System

This project is an IoT based smart motor protection system built using ESP32.

The system monitors motor temperature and load conditions. 
If the temperature or load exceeds safe limits, the motor automatically stops and alerts are triggered.

## Features
- Temperature monitoring
- Motor overload protection
- LED and buzzer alerts
- LCD display monitoring
- IoT dashboard integration

## Components Used
- ESP32
- Temperature Sensor (DS18B20)
- Servo Motor
- Relay Module
- LCD Display
- Buzzer
- LED
- Resistors
- Jumper Wires

## Tools Used
- Arduino IDE
- Wokwi Simulator
